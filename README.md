## Installation

- For self-hosting use following commands:

```
git clone https://github.com/Solamil/marlen
cd marlen/
go run main.go
```

## Dependencies

- `golang` >= 1.18 version
- `github.com/hashicorp/golang-lru` - LRU cache
- `https://github.com/beevik/etree` - XML parser
